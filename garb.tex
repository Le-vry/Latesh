# Berakna boxplot-statistik for LaTeX
def boxplot_stats(data, box_n=1):
    q1, median, q3 = np.percentile(data, [25, 50, 75])
    iqr = q3 - q1
    lower_w = max(min(data), q1 - 1.5*iqr)
    upper_w = min(max(data), q3 + 1.5*iqr)
    outliers = [x for x in data if x < lower_w or x > upper_w]
    
    print(f"{median:.1f}")
    print(f"Q1: {q1:.1f}", f"Q3: {q3:.1f}")
    print(f"lower whisker: {lower_w:.1f}")
    print(f"upper whisker: {upper_w:.1f}")
    
    # Formatera outliers for LaTeX (box_n: 1 for fore, 2 for efter)
    if outliers:
        coords = " ".join([f"({box_n},{x:.0f})" for x in outliers])
        print(f"outliers: {coords}")
    else:
        print("Inga outliers")

# Berakna for dina data
print("\\nBoxplot statistik - Fore:")
boxplot_stats(baseline_data, 1)
print("\\nBoxplot statistik - Efter:")
boxplot_stats(improved_data, 2)
\end{lstlisting}

I denna del ska du diskutera och utvärdera dina resultat i relation till syftet

\subsection{Utvärdering av förbättringens effektivitet}

Baserat på resultaten kan det konstateras att metod Y ledde till en statistiskt
signifikant och praktiskt relevant förbättring av egenskap X för produkttyp Z.

\textbf{Förbättringens storlek:}
Den uppmätta förbättringen på 23.6\% kan betraktas som substantiell inom detta
område. Jämfört med tidigare studier på liknande produkter\cite{waldspurger02}
ligger detta resultat i linje med förväntningar för denna typ av optimering.

\textbf{Statistisk tillförlitlighet:}
Med p < 0.001 och en effektstorlek (Cohen's d) på 2.74 visar resultaten på en
stark och tillförlitlig effekt. Det stora antalet mätningar (n=35) och den låga
standardavvikelsen stärker tillförlitligheten i resultaten.

\subsection{Lämplighet av metod Y för produkttyp Z}

\textbf{Fördelar:}
\begin{itemize}
    \item Betydande prestandaförbättring utan negativ påverkan på funktionalitet
    \item Relativt enkel implementering med begränsad kodomstrukturering
    \item Kompatibel med befintlig arkitektur
    \item Låg risk för introduktion av nya buggar
\end{itemize}

\textbf{Nackdelar och begränsningar:}
\begin{itemize}
    \item Ökad minnesanvändning kan vara problematisk för resursbegränsade
    enheter
    \item Implementering kräver fördjupad kunskap inom området
    \item Underhållskostnader kan öka på grund av ökad komplexitet
\end{itemize}

\subsection{Metodkritik och begränsningar}

Bedöm metodens lämplighet för att besvara frågorna: Var den effektiv och var det
något som kan förbättras? Diskutera vad som fungerade bra, vad som kunde gjorts
annorlunda och om något saknades.

\textbf{Testmiljöns representativitet:}
Mätningarna genomfördes i en kontrollerad miljö som kanske inte helt återspeglar
verkliga användningsförhållanden.

\textbf{Generaliserbarhet:}
Resultaten är specifika för den testade produkttypen och implementeringen.
Generalisering till andra produkttyper bör göras med försiktighet.

\subsection{Praktiska implikationer}

Jämför dina resultat med tidigare forskning eller liknande arbeten och notera
skillnader och likheter.

\textbf{Rekommendationer för implementation:}
Baserat på studiens resultat rekommenderas metod Y för liknande produkter av typ
Z, särskilt när prestandaförbättring är prioriterad över minnesbegränsningar.

\textbf{Förslag på framtida forskning:}
Ge rekommendationer för framtida studier och förbättringar:
\begin{itemize}

    \item Användarstudier för att utvärdera den upplevda förbättringen
\end{itemize}

\subsection{Slutsats}

Studien visar att metod Y är en lämplig och effektiv metod för att förbättra
egenskap X för produkttyp Z. Den statistiskt signifikanta förbättringen på
23.6\%, kombinerat med begränsade nackdelar, stöder implementering av denna
metod i liknande sammanhang.
